Конфигурационный файл config.xml отвечает за те параметры веб-приложения, которые могут быть определены пользователем, например: заголовок, состав слоев, состав виджетов. 
Файл config.xml расположен в одном каталоге с головным файлом веб-приложения – index.html и не должен быть перемещен без внесения соответствующих правок в код веб-приложения.
Конфигурационный файл начинается тэгом <configuration> и заканчивается тэгом </configuration>, между этими тэгами размещено содержимое конфигурационного файла.
Содержимое конфигурационного файла состоит из секций.
Каждая секция состоит из других секций или параметров.
Названия всех секций и параметров являются регистрозависимыми.
Все параметры задаются в кавычках.


<desing> - 	Секция оформления главной страницы приложения. Необязательная секция.
			Содержит заголовок страницы, цветовую тему и «шапку» веб-приложения.

			Параметры:
				title – заголовок страницы. Необязательный параметр. Значение по умолчанию – "JavaScript Viewer for ArcGIS".
				theme – цветовая тема страницы. Необязательный параметр. Возможные значения: "blue", "gray", "green", "orange", "purple". Значение по умолчанию – "gray".
			
				<heading> – секция «шапки» веб-приложения. Необязательная секция. "Шапка" располагается в верхней части браузера в виде полосы фиксированной высоты от левого края экрана до правого.
				Параметры:
					type – тип шапки. Необязательный параметр. может принимать два значения: "image" (шапка будет сделана с помощью вставки изображения) и "caption" (шапка будет сделана с помощью текстового оформления). Значение по умолчанию – "image".
					height – высота шапки в пикселах. Необязательный параметр. Значение по умолчанию – "63px".
					image – в случае, если тип шапки – изображение, ссылка на используемое изображение (обычно указывается относительный путь на веб-сервере). Необязательный параметр. Значение по умолчанию – "images/header.png"
					background – в случае, если тип шапки – изображение, в параметре задается ссылка на дополнительное изображение, которое заполнить полосу по горизонтали до необходимой ширины экрана. Необязательный параметр. Значение по умолчанию – "images/background.png".
					caption – в случае, если тип шапки – текст, в параметре задается текст для вывода. Необязательный параметр. Значение по умолчанию – "JavaScript Viewer for ArcGIS".
					font-size – в случае, если тип шапки – текст, в параметре задается размер шрифта в точках. Необязательный параметр. Значение по умолчанию – "35pt".
					font-name – в случае, если тип шапки – текст, в параметре задается название шрифта. Необязательный параметр. Значение по умолчанию – "Arial".
					font-color – в случае, если тип шапки – текст, в параметре задается цвет шрифта в шестнадцатеричном формате. Необязательный параметр. Значение по умолчанию – "#FFFFFF".
					background-color – в случае, если тип шапки – текст, в параметре задается цвет фона в шестнадцатеричном формате. Необязательный параметр. Значение по умолчанию = "#0000FF".
					text-align – в случае, если тип шапки – текст, в параметре выравнивание текста ("left" – пол левому краю, "center" - по центру, "right" – по правому краю). Необязательный параметр. Значение по умолчанию - "left".

<portal> - 	Секция параметров авторизации OAuth-2 в ArcGIS Online / Portal for ArcGIS. Необязательная секция.

			Параметры:
				url – URL портала. Необязательный параметр. Значение по умолчанию - http://www.arcgis.com.
				ssl - признак того, что соединение с порталом для авторизации будет проходить по протоколу SSL (другими словами, поддерживает ли портал SSL). Необязательный параметр. Возможные значения: "true" и "false". По умолчанию - "true.
				appID – идентификатор приложения в ArcGIS Online / Portal for ArcGIS. Обязательный параметр.
				expiration – время действия токена, выданного приложению, в минутах. Необязательный параметр. Значение по умолчанию - "60".
				
<proxy> - 	Секция настройки прокси-страницы. Необязательная секция.

			Параметры:
				useProxy - параметр указывает, использовать или нет прокси-страницу. Необязательный параметр. Возможные значения: "true" и "false". По умолчанию - "false".
				proxyUrl – параметр указывает путь к прокси-странице на веб-сервере. Необязательный параметр. Значение по умолчанию – "proxy.ashx".

<tasks> - 	Секция описания аналитических сервисов. Необязательная секция. Аналитические сервисы используются в виджете адресного поиска, редактирования, построения маршрутов, экспорта карты.

			<geometry> – секция описания сервиса геометрии. Необязательная секция.
			Параметры:
				url – REST URL сервиса геометрии. Необязательный параметр. Значение по умолчанию – "http://tasks.arcgisonline.com/arcgis/rest/services/Geometry/GeometryServer" (сервис геометрии ArcGIS Online).

			<geocoder> – секция описания сервиса геокодирования. Необязательная секция.
			Параметры:
				url – REST URL сервиса геокодирования. Необязательный параметр. Значение по умолчанию –  "http://geocode.arcgis.com/arcgis/rest/services/World/GeocodeServer" (сервис геокодирования ArcGIS Online).
				singleLineFieldName – имя поля для однострочного геокодирования.  Необязательный параметр. Значение по умолчанию – "SingleLine". 

			<route> – секция описания сервис сетевого анализа (построение маршрутов). Необязательная секция.
			Параметры:
				url – REST URL сервиса построения маршрута. Необязательный параметр. Значение по умолчанию -  "http://route.arcgis.com/arcgis/rest/services/World/Route/NAServer/Route_World" (сервис построения маршрута ArcGIS Online).

			<print> – секция описания сервиса экспорта карты в веб-файл PDF или PNG. Необязательная секция.
			Параметры:
				url - REST URL сервиса экспорта карты. Необязательный параметр. Значение по умолчанию - "http://sampleserver6.arcgisonline.com/arcgis/rest/services/Utilities/PrintingTools/GPServer/Export%20Web%20Map%20Task" (сервис экспорта карты ArcGIS Online).

				
<dijits> -	Секция виджетов. Необязательная секция. Секцмя состоит из одной или нескольких секций [dijit]. Каждая секцмя <dijit> определяет виджет, который будет использован в веб-приложении.
			Секция <dijit> содержит параметр type, определяющий тип виджета, а также другие параметры, специфичные для каждого конкретного виджета.
			Доступны следующие типы виджетов: редактор (editor), идентификация объектов на карте (identify), измерения по карте (measure), поиск объектов по адресу (geocoder), построение маршрутов (route), экспорт карты в PDF и PNG (print),
			список слоев и легенда (toc), закладки (bookmarks), информация (info), переключтель базовой карты (basemaps), фильтр по дате (dateFilter), линейка масштабов (scalebar), обзорное окно карты (overview).
			Включение секции виджета в конфигурационный файл означает появление виджета в приложении.
			
			<dijit type = "editor"> - секция виджета редактирования. Необязательная секция.
			Параметры:
				toolbar – наличие панели инструментов. Необязательный параметр. Возможные значения: "true", "false". Значение по умолчанию - "true".
				
				<tools> – секция описания состава панели инструментов. Необязательная секция.
				Параметры:
					undo-redoTool – инструмент отмены и повтора действия. Необязательный параметр. Возможные значения – "true" и "false". Значение по умолчанию – "true".
					undo-redoMaxOperations – максимальное число операций в буфере undo-redoTool. Необязательный параметр. Значение по умолчанию – "10".
					mergeTool – инструмент слияния фигур. Необязательный параметр. Возможные значения – "true" и "false". Значение по умолчанию – "true".
					cutTool – инструмент разрезания фигур. Необязательный параметр. Возможные значения – "true" и "false".Значение по умолчанию – "true".
					reshapeTool – инструмент изменения формы фигур. Необязательный параметр. Возможные значения – "true" и "false". Значение по умолчанию – "true".
					polygonDrawTools – инструменты рисования полигонов. Перечисляются через запятую. Необязательный параметр. Значение по умолчанию – все возможные инструменты – "POLYGON,FREEHAND_POLYGON,AUTOCOMPLETE,RECTANGLE,CIRCLE,TRIANGLE,ELLIPSE".
					polylineDrawTools – инструменты рисования полилиний. Перечисляются через запятую. Необязательный параметр. Значение по умолчанию – все возможные инструменты – "POLYLINE,FREEHAND_POLYLINE,AUTOCOMPLETE,RECTANGLE,CIRCLE,TRIANGLE,ELLIPSE".

			<dijit type = "identify"> - секция виджета идентификации объектов на карте. Необязательная секция.
			Параметры:
				tolerance – допуск (в пикселах) погрешности выбора объекта на карте. Необязательный параметр. Значение по умолчанию – "10".
			
			<dijit type = "measure"> - секция виджета измерений по карте. Необязательная секция.

			<dijit type = "geocoder"> - секция виджета адресного поиска. Необязательная секция.
			
			<dijit type = "route"> - секция виджета построения маршрутов. Необязательная секция.
			
			<dijit type = "print"> - секция виджета экспорта карты в PDF и PNG. Необязательная секция.
			Параметры:
				title – заголовок документа. Необязательный параметр. Значение по умолчанию – "JavaScript Viewer for ArcGIS".
				owner – владелец документ. Необязательный параметр. Значение по умолчанию – "Esri CIS".
			
			<dijit type = "toc"> - секция виджета списка слоев и легенды. Необязательная секция.
			
			<dijit type = "bookmarks"> - секция виджета закладок. Необязательная секция.
			Параметры:
				editable – разрешено ли пользователю редактировать закладки. Необязательный параметр. Возможные значения – "true" и "false". Значение по умолчанию – "true".
				
				<bookmarks> – секция списка закладок. Необязательная секция. Список состоит из секций <bookmark>.
					<bookmark> - секция описания закладки. Необязательная секция.
					Параметры:
						title – название закладки. Обязательный параметр.
						
						<extent> – секция экстента закладки. Обязательная секция.
						Параметры:
							xmin - координата левого края экстента. Обязательный параметр.
							ymin - координата верхнего края экстента. Обязательный параметр.
							xmax - координата правого края экстента. Обязательный параметр.
							ymax - координата нижнего края экстента. Обязательный параметр.
							wkid - идентификатор системы координат экстента. Обязательный параметр. 
			
			<dijit type = "info"> - секция виджета информации. Необязательная секция.
			Параметры:
				html – html-строка, описывающая содержимое панели виджета. Необязательный параметр. Значение по умолчанию: "Разработка компании &lt;a href = &quot;http://www.esri-cis.ru&quot; target = &quot;_blank&quot; &gt;Esri CIS&lt;/a&gt;.&lt;/br&gt;&lt;/br&gt; Версия 1.0 (сборка 05.07.2013).&lt;/br&gt;&lt;/br&gt;&lt;a href = &quot;help.docx&quot; target = &quot;_blank&quot; &gt;&lt;B&gt;Загрузить справку.&lt;/&gt;&lt;/a&gt;"

			<dijit type = "basemaps"> - секция виджета переключения базовой карты. Необязательная секция.
			Параметры:
				arcgismaps - включить карты ArcGIS Online в галерею карт. Необязательный параметр. Возможные значения: "true" и "false" . Значение по умолчанию – "true".
				googlemaps – включить карты Google Maps в галерею карт. Необязательный параметр. Возможные значения: "true" и "false" . Значение по умолчанию – "false".

			<dijit type = "dateFilter"> - секция виджета фильтра объектов карты по дате. Необязательная секция.
			Параметры:
				limits – тип границ фильтра. Необязательный параметр. Возможные значения – "single" (точно по введенной дате) и "double" (границы – от..до). Значение по умолчанию – "single".
				field – поле, из которого будет взять значение для фильтрации объектов. Общее значение для всех слоев. Обязательный параметр.

			<dijit type = "scalebar"> - секция виджета линейки масштабов. Необязательная секция.
			
			<dijit type = "overview"> - секция виджета обзорного окна карты. Необязательная секция.

<map> -  	Cекция карты. Обязательная секция.
			Содержит состав и описание слоев карты.
			Включает следующие секции: <options> (параметры карты), <basemaps> (базовые карты), <mapLayers> (слои).

			<options> – секция параметров карты. Необязательная секция.
			Параметры:

				wkid – идентификатор системы координат. Необязательный параметр. Если параметр не указан, карта примет систему координат первого добавленного на карту слоя.
				slider – инструмент масштабирования. Необязательный параметр. Возможные значения "small" (крупный), "large" (мелкий) и "none" (инструмент не отображается). Значение по умолчанию – "small".
				esriLogo – логотип Esri в правом нижнем углу экрана. Необязательный параметр. Возможные значения – "true" и "false". Значение по умолчанию – "true".
				attribution – указание источников карты в правом нижнем углу. Необязательный параметр. Возможные значения – "true" и "false". По умолчанию – "true".
				basemap – базовая карта, которая будет отображена при загрузке карты. Необязательный параметр. В параметре указывается идентификатор базовой карты (может быть идентификатор базовой карты в секции <basemaps>.<basemap> либо один из идентификаторов базовой карты ArcGIS Online: "streets" , "satellite" , "hybrid", "topo", "gray", "oceans", "national-geographic", "osm"). Если идентификатор ссылается на базовую карту из секции <basemaps>.<basemap>, и эта карта по каким-то причинам недоступна, то будет использована карта OpenStreetMap (идентификатор 'osm' из ArcGIs Online). При отсутствии параметра базовая карта отображена не будет, если только она не задана с помощью слоев в секции <mapLayers>.
				
				<extent> - секция для определения начального экстента карты. Необязательная секция. Если секция не указана, карта примет начальный экстент первого добавленного на карту слоя.
				Параметры:
					xmin - координата левого края экстента. Обязательный параметр.
					ymin - координата верхнего края экстента. Обязательный параметр.
					xmax - координата правого края экстента. Обязательный параметр.
					ymax - координата нижнего края экстента. Обязательный параметр.
				
			<basemaps> – секция, в которой перечисляются базовые карты, которые будут включены в галерею базовых карт виджета выбора базовой карты. Необязательная секция. Секция внутри себя содержит одну или несколько секций <basemap>.
				<basemap> - секция описания базовой карты. Необязательная секция.
				Параметры:
					title – название базовой карты. Обязательный параметр. Пример: "Базовая карта".
					image - изображение базовой карты. Обязательный параметр. Указывается абсолютный путь к файлу или относительный путь. Пример: "images/basemaps/basemap_rosreestr.png".
					id – идентификатор базовой карты. Обязательный параметр. Пример "rosreestr".

					<layers> - секция, в которой перечисляются слои базовой карты. Обязательная секция. Секция внутри себя содержит одну или несколько секций <layer>.
						<layer> - секция описания слоя базовой карты. Обязательная секция.
						Параметры:
							type – тип слоя. Возможные значения: "tiled", "dynamic". Обязательный параметр.
							url – REST-url сервиса, передающего слой. Обязательный должен быть указан либо этот параметр, либо portalID. Если указаны оба, применяется url.
							portalD - идентификатор слоя как ресурса в ArcGIS Online или Portal for ArcGIS. Обязательно должен быть указан либо этот параметр, либо url. Если указаны оба, применяется url.
							id – идентификатор слоя. Необязательный параметр.
							opacity – прозрачность слоя. Необязательный параметр. Значения от 0 до 1. Значение по умолчанию – "1" (полностью непрозрачен).
							token – токен. Необязательный параметр.

			<mapLayers> - секция, в которой перечисляются слои карты. Обязательная секкиця. Секция внутри себя содержит одну или несколько секций <layer>.
				<layer> - секция описания слоя карты (или группы слоев карты, передаваемый через общий динамический сервис). Обязательная секция.
				Параметры:
					type - тип слоя. Обязательный параметр. Возможные значения: "tiled" (тайловый), "dynamic" (динамический), "feature" (слой объектов).
					id - идентификатор слоя. Необязательный параметр.
					title - заголовок слоя. Необязательный параметр. По умолчанию берется из настроек сервиса.
					url - REST-url слоя. Обязательно должен быть указан либо этот параметр, либо portalID.
					portalID - идентификатор слоя как ресурса в ArcGIS Online или Portal for ArcGIS. Обязательно должен быть указан либо этот параметр, либо url.
					visible - видимость слоя по умолчанию. Необязательный параметр. Возможные значения - "true" и "false". Значение по умолчанию - "true".
					opacity – прозрачность слоя. Необязательный параметр. Значения от 0 до 1. Значение по умолчанию – "1" (полностью непрозрачен).

Возможные значения: "ONDEMAND" (объекты запрашиваются с сервера для видимой области экстента), "SNAPSHOT" (при загрузке слоя с сервера запрашиваются все объекты), "SELECTION" (объекты запрашиваются по запросу).
